# Spec 01 — Auth & Gmail Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Google sign-in via Better Auth (mounted inside Elysia at `/api/v1/auth`), backed by Drizzle/Postgres, plus the ability to grant the `gmail.readonly` scope and retrieve a usable Gmail access token server-side.

**Architecture:** Better Auth instance (`src/server/auth/auth.ts`) with the Google social provider and the Drizzle adapter, mounted into the single Elysia app via `new Elysia({name:"better-auth"}).mount(auth.handler)` and `.use()`'d into the `prefix:"/api/v1"` router. `basePath:"/api/v1/auth"` makes the mounted handler resolve correctly under the prefix (avoids the Elysia `.mount` prefix bug, since Better Auth's basePath equals the full external path). Auth tables live in `src/server/drizzle/schemas/auth-schema.ts` (generated by the Better Auth CLI). The Next catch-all (`app/api/[[...slugs]]/route.ts`) already forwards every `/api/*` request into Elysia, so auth + v1 API both flow through one server.

**Tech Stack:** better-auth 1.6.x, `@better-auth/drizzle-adapter`, `better-auth/plugins` (openAPI, multiSession), Elysia, Drizzle + postgres + pgvector, Gmail REST API (read-only).

**Spec:** `docs/superpowers/specs/2026-06-13-spec-01-auth-gmail.md` (this plan refines its auth-mount approach — see "Decisions" below).

```ts
import { drizzleAdapter } from "@better-auth/drizzle-adapter";
import { APIError, betterAuth } from "better-auth";
import { multiSession, openAPI } from "better-auth/plugins";
import { headers } from "next/headers";
import { cache } from "react";
import { ServerConfig } from "@/config/server-config";
import { db } from "@/server/drizzle/db";
import * as authSchema from "@/server/drizzle/schemas/auth-schema";
import { logger } from "@/server/logger";

export const auth = betterAuth({
    experimental: { joins: true },
    baseURL: ServerConfig.baseUrl,
    basePath: "/api/v1/auth",
    socialProviders: {
        google: {
            clientId: ServerConfig.google.clientId,
            clientSecret: ServerConfig.google.clientSecret,
            // Needed for a refresh token (offline Gmail access).
            accessType: "offline",
            prompt: "select_account consent",
        },
    },
    plugins: [openAPI(), multiSession()],
    database: drizzleAdapter(db, { provider: "pg", schema: authSchema }),
});

export const authenticate = cache(async () => {
    try {
        const session = await auth.api.getSession({ headers: await headers() });
        if (!session) return null;
        return { user: session.user, session: session.session };
    } catch (e) {
        if (e instanceof APIError) {
            logger.warn("auth APIError", e.message);
            return null;
        }
        logger.error("auth error", e);
        return null;
    }
});
```

### Task 5: Gmail scope helper (client) + token + Gmail service (server)

- [ ] **Step 1: Create `src/server/services/gmail.ts`**

```ts
import { auth } from "@/server/auth/auth";

export const GMAIL_READONLY_SCOPE =
    "https://www.googleapis.com/auth/gmail.readonly";

export interface GmailProfile {
    emailAddress: string;
    messagesTotal: number;
}

export async function getGoogleAccessToken(
    userId: string,
    requestHeaders: Headers,
): Promise<string> {
    const { accessToken } = await auth.api.getAccessToken({
        body: { providerId: "google", userId },
        headers: requestHeaders,
    });
    return accessToken;
}

export async function getGmailProfile(
    accessToken: string,
): Promise<GmailProfile> {
    const res = await fetch(
        "https://gmail.googleapis.com/gmail/v1/users/me/profile",
        { headers: { Authorization: `Bearer ${accessToken}` } },
    );
    if (!res.ok) {
        throw new Error(`Gmail API error ${res.status}`);
    }
    return (await res.json()) as GmailProfile;
}
```

```ts
import { and, eq } from "drizzle-orm";
import { Elysia } from "elysia";
import { auth } from "@/server/auth/auth";
import { db } from "@/server/drizzle/db";
import { account } from "@/server/drizzle/schemas/auth-schema";
import { GMAIL_READONLY_SCOPE } from "@/server/services/gmail";

export const meRouter = new Elysia().get("/me", async ({ request }) => {
    const session = await auth.api.getSession({ headers: request.headers });
    if (!session) {
        return { user: null, gmailConnected: false };
    }
    const rows = await db
        .select({ scope: account.scope })
        .from(account)
        .where(
            and(
                eq(account.userId, session.user.id),
                eq(account.providerId, "google"),
            ),
        );
    const gmailConnected = rows.some((r) =>
        r.scope?.includes(GMAIL_READONLY_SCOPE),
    );
    return { user: session.user, gmailConnected };
});
```
